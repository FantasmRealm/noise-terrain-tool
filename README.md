# Noise Terrain Tool 🎮

一款基于噪波图的3D地形高度图生成工具，帮助游戏开发者快速构建随机地形场景。

---

## ✨ 功能特点

- **噪波地形生成**：通过生成黑白噪波地形图实现高度映射，白色（灰度255）代表最高地形，黑色（灰度0）代表平地
- **参数自由调整**：支持自定义地形高度缩放、平面缩放（X/Z轴）等参数
- **多格式导出**：可导出OBJ、GLB等主流3D模型格式，直接导入Godot等游戏引擎
- **轻量化实现**：纯HTML+JavaScript实现，无需安装，浏览器直接运行

## 📦 本地部署（离线使用）

1.  点击页面右上角的绿色 **「Code」** 按钮，在下拉菜单中选择 **「Download ZIP」**，将项目压缩包下载到本地。
2.  找到下载的 ZIP 文件，右键选择解压到任意文件夹。
3.  进入解压后的文件夹，找到 `TerrainTool.html` 文件，双击打开即可在浏览器中使用工具。

---

## 🚀 快速开始

1.  打开项目中的 `TerrainTool.html` 文件，进入工具主页。
2.  **高度图生成（可选）**：
    -   切换到「高度图生成」标签页。
    -   设置图片宽度、高度、噪波缩放和细节层数。
    -   点击「生成高度图」预览灰度图，再点击「下载PNG高度图」保存文件。
3.  **模型生成**：
    -   切换到「模型生成（识别高度图）」标签页。
    -   点击「选择文件」，上传黑白/灰度PNG高度图（或直接使用内置噪波生成的图）。
    -   调整「地形高度缩放」（值越大，地形越高）和「平面缩放（X/Z轴）」参数。
    -   选择导出模型格式（推荐OBJ，通用兼容性好）。
4.  点击「生成并下载3D模型」，等待处理完成后即可得到可直接使用的地形模型。

⚠️ 注意：生成的模型默认不包含材质信息，导入到游戏引擎后，需要手动配置材质和纹理。

---

## 📐 核心规则与Godot使用说明

- 白色区域（灰度255）= 最高地形（草地/山峰），黑色区域（灰度0）= 最低地形（平地/洼地）
- 生成的模型可直接拖拽到Godot等游戏引擎，会自动识别网格体
- 推荐尺寸：256x256（平衡性能和细节），避免使用2048x2048超大尺寸，会导致模型加载卡顿
