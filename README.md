# Noise Terrain Tool 🎮

一款基于噪波图的3D地形高度图生成工具，帮助游戏开发者快速构建随机地形场景。

---

## ✨ 功能特点

- **噪波地形生成**：通过生成黑白噪波地形图实现高度映射，白色（灰度255）代表最高地形，黑色（灰度0）代表平地
- **参数自由调整**：支持自定义地形高度缩放、平面缩放（X/Z轴）等参数
- **多格式导出**：可导出OBJ、GLB等主流3D模型格式，直接导入Godot等游戏引擎
- **轻量化实现**：纯HTML+JavaScript实现，无需安装，浏览器直接运行

---

## 🚀 快速开始

1. 打开项目中的 `terrain高度图生成.html` 文件
2. 点击“选择文件”上传高度图（或使用内置噪波生成）
3. 调整“地形高度缩放”和“平面缩放”参数
4. 选择导出格式（OBJ/GLB）
5. 点击“生成并下载3D模型”，即可得到可直接使用的地形模型

> ⚠️ 注意：生成的模型默认不包含材质信息，导入到游戏引擎后，需要手动配置材质和纹理。

---

## 📐 核心规则与Godot使用说明

- 白色区域（灰度255）= 最高地形（草地/山峰），黑色区域（灰度0）= 最低地形（平地/洼地）
- 生成的模型可直接拖拽到Godot场景，MeshInstance3D会自动识别
- 推荐尺寸：256x256（平衡性能和细节），避免使用2048x2048超大尺寸，会导致模型加载卡顿
